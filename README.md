# Huawei LTE Router SMS to E-mail Sender

The python script can help you read your sms from e-mail box.

Tested on:
* HUAWEI B525s-65a

## How to use

1. copy .env.example to .env
```console
$ cp .env.example .env
$ vim .env
```

2. just run it!
```console
$ python3 check-sms.py
```
